Refurb-it

Refurb-it seeks to refurbish secondhand furniture with creative and hand-picked artists to make old become new and unique. We strive to produce high quality products with the fraction of the price of major retailers while giving the customer a one-of-a-kind product.

Overview

The project assigned was to create an Ecommerce responsive website that was hitting a target item that was hard to find in regular stores. Students were allowed all creative control and brand guidelines.

User Stories

We received over twelve user stories to select three from. Below are the three stories chosen for this project.

  • As a new customer, I want to access the inventory without having to register, so that I can make sure this store has what Iâ€™m looking for before having to create an account.
  • As a returning customer, I want to be able to save item that I cannot buy to a wish list, so that I can purchase them at a later date.
  • As a returning customer, I want to see an overview of my previous activity, so that I can view previous purchases and search history.

Style Guide

My goal for the style of this project was to keep it modern and sleek and focus more on the images of the products. Photos that are used are staged as the item is likely used, with bright lightening.
